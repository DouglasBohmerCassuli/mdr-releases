# MDR Releases

Repositorio publico usado para publicar instaladores e pacotes de atualizacao do MDR.

Para instalar o programa:

1. Vá até a última versão de release
2. Faça oo download do "MDR-Setup-v1.0.X.exe"
3. Após baixar o arquivo basta roda-lo!
4. O sistema irá baixar novas atualizações de forma automática
