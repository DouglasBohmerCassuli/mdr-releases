# MDR Releases

Repositório publico usado para publicar instaladores e pacotes de atualização do MDR.

Como instalar o programa:

1. Vá até a última versão de release;
2. Faça o download do "MDR-Setup-vX.X.X.exe";
3. Após baixar o arquivo basta roda-lo;
4. O sistema irá baixar novas atualizações de forma automática.
